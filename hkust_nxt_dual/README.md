# HKUST NXT-Dual 研究目录

- `目录关系.md`：nxt-dual 板目录映射与 build 依赖示意。
- `与公版对比.md`：记录与 PX4 公版固件的差异。
- `USB与CDC说明.md`：USB CDC/NuttShell、MAVLink、uXRCE 之间的关系。
- `uXRCE端口.md`：当前版本的 uXRCE-DDS 串口配置与调试步骤。
- `串口映射.md`：`/dev/ttyS*` 与 MCU UART 的对照。
- `USBC改NSH教程.md`：如何将 USB CDC 改成 NSH 控制台。

本目录持续补充 HKUST 板的专有外设、脚本和参数默认值，所有修改完成后请同步更新此文件。
